============
Installation
============


Stable release
--------------

To install feature-selection, run this command in your terminal:

.. code-block:: console

    $ pip install -i https://test.pypi.org/simple/ feature-selection

This is the preferred method to install feature-selection, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for feature-selection can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/UBC-MDS/feature-selection

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/UBC-MDS/feature-selection/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/UBC-MDS/feature-selection
.. _tarball: https://github.com/UBC-MDS/feature-selection/tarball/master
